__Polecenia:__

> _bundle init_ - utworzenie pliku Gemfile;  
> _bundle install_ - instalacja pakietów z pliku Gemfile;  
> _bundle exec rspec_  - uruchomienie wszystkich testów z plików '*_spec.rb' z katalogu spec.
